# Fake News Detection System

A Machine Learning based web application developed to automatically detect and classify news articles as Real or Fake. The system uses Natural Language Processing (NLP) and multiple ML algorithms to analyze news content with high accuracy.

## Features
- Real-time Fake News Detection
- Confidence Score Display
- URL and Domain Analysis
- Clickbait Detection
- Feature Analysis Dashboard
- REST API Endpoint
- Responsive User Interface
- Support for Multiple News Sources

## Technologies Used
- Python
- Flask
- Scikit-learn
- NLTK
- Pandas
- NumPy
- HTML
- CSS

## Model Performance
- Best Model: Gradient Boosting Classifier
- Accuracy: 97.79%
- AUC-ROC: 0.9950
- F1-Score: 0.9779

## Installation
1. Clone the repository
2. Install dependencies:
pip install -r requirements.txt
3. Train the models:
python load_data.py
python preprocess.py
python feature_extraction.py
python train_model.py
4. Run the application:
python app.py
